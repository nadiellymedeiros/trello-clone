# Descrição

Esse projeto foi como treinamento pessoal. Consiste em uma aplicação web de um organizador de tarefas. Podemos adicionar tarefas, movimentar a tarefa entre as colunas TO DO, DOING, DONE.

# Features

### Frontend (Telas)

- [x] Card To do
- [x] Card Doing
- [x] Card Done
- [x] Add Card
